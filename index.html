Project Overview

This case study walks through the redesign of MCDONALD'S food delivery APP with the goal of improving user engagement and retention by optimizing the app's interface and user journey.

UX Design Process

1. Research

User interviews, surveys, competitor analysis

2. Define

Persona creation, problem statements

3. Ideate

Sketches, wireframes, journey maps

4. Prototype

Clickable prototypes using Figma

5. Test

Usability testing an iterations
